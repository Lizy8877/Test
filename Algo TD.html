Exercice 1

Construire un algorithme qui nous calcule le Nième (avec N>2) terme de la suite de FIBONACCI qui est définie par : Suite de FIBONACCI : 1, 1, 2, 3, 5, 8, 13, .......


def fibonacci(n): if n <= 1: return 1 a, b = 1, 1 for _ in range(2, n): a, b = b, a + b return b # Demander à l'utilisateur un nombre try: n = int(input("Entrez un nombre pour calculer la suite de Fibonacci : ")) if n < 0: print("Veuillez entrer un nombre entier positif.") else: resultat = fibonacci(n) print(f"Le {n}ᵉ nombre de Fibonacci est : {resultat}") except ValueError: print("Veuillez entrer un nombre entier valide.") fibonacci(n)



Exercice 2

Sachant qu'un nombre premier est un nombre qui n'accepte aucun diviseur excepté 1 et lui- même. Construire la solution qui nous permet de savoir si un nombre est premier ou non.


def est_premier(n): if n <= 1: return False for i in range(2, int(n**0.5) + 1): if n % i == 0: return False return True # Demander à l'utilisateur un nombre à vérifier try: n = int(input("Entrez un nombre pour vérifier s'il est premier : ")) if est_premier(n): print(f"Le nombre {n} est un nombre premier.") else: print(f"Le nombre {n} n'est pas un nombre premier.") except ValueError: print("Veuillez entrer un nombre entier valide.") est_premier(n)



Exercice 3

Sachant qu'il n'existe que 4 nombres compris entre 100 et 500 tels que la somme des cubes des chiffres les composant est égale au nombre lui-même. Construire la solution qui permet de retrouver ces 4 nombres.


def est_nombre_magique(n): # Extraire les chiffres du nombre chiffres = [int(chiffre) for chiffre in str(n)] # Calculer la somme des cubes des chiffres somme_cubes = sum(chiffre**3 for chiffre in chiffres) # Vérifier si la somme des cubes est égale au nombre lui-même return somme_cubes == n # Liste pour stocker les nombres qui satisfont la condition nombres_magiques = [] # Parcourir tous les nombres entre 100 et 500 for nombre in range(100, 501): if est_nombre_magique(nombre): nombres_magiques.append(nombre) # Afficher les résultats print("Les nombres magiques entre 100 et 500 sont :") print(nombres_magiques)



Exercice 4

Un nombre parfait est un nombre qui est égal à la somme de tous ses diviseurs excepté lui- même. Rechercher tous les nombres parfaits compris entre 1 et N. Exemple de nombre parfait : 6


def est_parfait(nombre): # Trouver tous les diviseurs de nombre sauf lui-même diviseurs = [i for i in range(1, nombre) if nombre % i == 0] # Vérifier si la somme des diviseurs est égale au nombre return sum(diviseurs) == nombre # Demander à l'utilisateur de saisir la valeur de n try: n = int(input("Entrez un nombre (N) pour trouver tous les nombres parfaits entre 1 et N : ")) if n < 1: print("Veuillez entrer un nombre entier positif.") else: # Rechercher tous les nombres parfaits entre 1 et n nombres_parfaits = [i for i in range(1, n + 1) if est_parfait(i)] # Afficher les résultats print(f"Les nombres parfaits entre 1 et {n} sont : {nombres_parfaits}") except ValueError: print("Veuillez entrer un nombre entier valide.")

Exercice 8

Construire la solution qui effectue un swapping, autrement dit qui échange lesoctets de poids fort et de poids faible d'un nombre entier quelconque.


Partie qui s'en rapproche le plus
def swapping(n): s = str(abs(n)) swapped = int(s[::-1]) * (-1 if n < 0 else 1) return swapped



Exercice 9

A partir d’un nombre entier N on voudrait obtenir deux autres nombres N1 et N2. Le premier (N1) sera constitué par les chiffres pairs de N et le second (N2) par les chiffres impairs.Exemples :


def separer_chiffres(n): pairs = ''.join(c for c in str(n) if int(c) % 2 == 0) impairs = ''.join(c for c in str(n) if int(c) % 2 != 0) return int(pairs or 0), int(impairs or 0) # Demander à l'utilisateur de saisir un nombre n = int(input("Veuillez entrer un nombre : ")) pairs, impairs = separer_chiffres(n) print(f"Chiffres pairs : {pairs}") print(f"Chiffres impairs : {impairs}")
